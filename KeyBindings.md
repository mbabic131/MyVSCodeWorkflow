# Navigate

Crtl+P => Go to file

Ctrl+R => Go to symbol 

Ctrl+Shift+R => Go to symbol in project

Ctlr+, => Go to settigns


# Files

Ctrl+Shift+O => Open folder (project)

Ctrl+K+S => Keyboard mapping

Ctrl+Alt+N => New file

Ctr+K+B => Show/Hide File Explorer

Ctrl+Shift+X => Show/Hide Extensions
